# Tragamonedas 3x5 con Tkinter

Un juego interactivo de tragamonedas con patrones de premios y animación.

## Características

- Interfaz gráfica con Tkinter.
- Sistema de premios avanzados (jackpot, figuras, líneas).
- Animación de giro.
- Probabilidades ajustadas por símbolo.

## Cómo ejecutar

```bash
python run.py
